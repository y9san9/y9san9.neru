# neru

[Neru](https://github.com/y3owk1n/neru) is a free and open-source software that
makes it possible to get rid of your mouse while using operating system. It
supports a lot of things and I can't fit them all in there, but mainly it's 2
modes that are interesting:

- Hints
- Recursive Grid

Hints basically do Vimium from your entire operating system (which is what I
started to use, but I quickly realized that I will not use it in the long run).

Recursive grid is what I use now. There are so many configuration options, so
you can play around with it and my config is just one of the ways you can
configure it. My config is very different from the default one.

It has 25 keys (5x5 grid), but it makes it possible to click almost everywhere
with 2 clicks and everywhere "for sure" with 3 clicks. My primary shortcut does
auto-click when max zoom level is reached, so I don't need to press "Enter +
Escape" which is 2 more keys (40% difference in my case).

After all I use it like Vimium, but for the entire OS and clicks are consistent
with element position on screen. So it's much better for me.
